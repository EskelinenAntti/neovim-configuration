# Neovim configuration

This is my modest Neovim configuration.

It includes:

- LSP configuration with native completion
- file and text search with `fzf-lua`
- formatting with `conform.nvim`
- file browsing with `oil.nvim`

## Installation

The setup is intended for macOS and uses Homebrew:

```sh
git clone https://github.com/EskelinenAntti/neovim-configuration.git ~/.config/nvim
cd ~/.config/nvim
brew bundle
```

Install neovim and other dependencies with `brew bundle`.
